super() 函数

def super(cls, inst):
    mro = inst.__class__.mro()
    return mro[mro.index(cls) + 1]
  • 通过 inst 生成 MRO 的 list
  • 通过 cls 定位当前 MRO 的 index，并返回 mor[index +1]

不要一说到 super 就想到父类！super 指的是 MRO 中的下一个类！
